## Getting Started

Follow these steps to run the project locally.

### 1. Clone the repository

```
git clone https://github.com/resyaaaaa/my-portfolio.git
```

### 2. Navigate to the project folder

```
cd path-to\my-portfolio
```

### 3. Install dependencies

```
npm install
```

### 4. Run the project

```
npm start
```

The app will run at:

```
http://localhost:3000
```
